techcloudup / Domain Takedown Investigator
GitHub
What this does: Enter a suspected phishing or brand-impersonation domain. The tool runs a full passive investigation — DNS records, WHOIS, SSL certificate, HTTP headers, robots.txt, and sitemap analysis — then outputs a structured report with all data needed to file abuse reports with the hosting provider, domain registrar, APWG, IC3, Google Safe Browsing, and FTC.
DNS / WHOIS → SSL Cert → HTTP Headers → Sitemap → Summary Report
This tool performs passive, read-only network queries (DNS, WHOIS, HTTP GET). It does not send any traffic to the target beyond standard HTTP requests.
